# Zerdeli Dashboard

Аналитикалық дашборд — Zerdeli International School желісі бойынша Сапа бақылау тесті (2026-05-04) нәтижелерінің талдауы.

🔗 **Live demo:** https://gabrielprogramist.github.io/zerdeli-dashboard/

## Не бар
- Желі деңгейі бойынша 7 мектепті салыстыру
- Әр мектептің ішкі талдауы (параллель сыныптар, әлсіз/күшті тақырыптар)
- Пән координаторларына арналған терең талдау (Математика, Ағылшын)
- Сүзгісіз «Big Picture» split-cell heatmap (мектеп × сынып × литер × пән)
- Толық мобильді бейімділік
